#  VantaCrew_task_2

A clean, fast, and minimal **Pipeline Tracker** built with React + Tailwind CSS. Track your prospects from first contact all the way to a closed deal — no backend required, everything saves locally in your browser.

## ✨ Features

-  **Add / Edit / Delete** prospects with a smooth modal form
-  **6 Pipeline Stages** — Lead → Contacted → Qualified → Proposal → Won → Lost
-  **Search** prospects by name, company, email, or notes
-  **Filter by Stage** with color-coded pill buttons
-  **Pipeline Bar** — visual breakdown of all stages at a glance
-  **Stats Dashboard** — Total prospects, Active deals, Closed Won value
-  **Persistent Storage** — data saved in `localStorage`, survives page refresh
-  **Keyboard friendly** — Escape key closes modals, auto-focus on open
-  **Responsive** — works on mobile, tablet, and desktop
